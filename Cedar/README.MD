# Cedar Policy Evaluation Example

This project demonstrates how to use **Cedar** to enforce **authorization policies** for an application that manages **bikes** and **cars** as resources. The policies define who can **view** or **update** a bike or car information based on their role

## Cedar Installation

Install Rust via rustup
`curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh` 

Install Cedar CLI via cargo 
`cargo install cedar-policy-cli`
